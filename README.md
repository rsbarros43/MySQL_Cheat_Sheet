# MySQL Database Cheat Sheet & Interactive DBA Lab

## 📖 Sobre o Projeto

O **MySQL Database Cheat Sheet & Interactive DBA Lab** é uma plataforma educacional desenvolvida para auxiliar estudantes, DBAs, Administradores de Sistemas, DevOps Engineers e profissionais de Banco de Dados no aprendizado e consulta rápida de comandos, conceitos e boas práticas do MySQL.

O projeto foi criado com foco em:

* Aprendizado prático
* Consulta rápida (Cheat Sheet)
* Simulador interativo de comandos
* Treinamento para DBAs
* Preparação para entrevistas técnicas
* Laboratórios de administração MySQL

## ⚡ Simulador Interativo

O projeto possui um simulador de comandos MySQL integrado diretamente ao navegador.

Exemplos:

```sql
show databases;
show tables;
show processlist;
show engines;
show variables;
show binary logs;
show replica status\G
select @@version;
status;
```

O simulador foi criado para fins educacionais e demonstra respostas realistas utilizadas por DBAs no dia a dia.
